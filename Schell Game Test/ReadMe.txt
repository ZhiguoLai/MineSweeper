How to open the game:
1.Open the folder named Build
2.Open the Schell Game Test.exe

How to play the game:
Use the mouse to play the game:
Left click to discover the grid.
Right click to mark the grid.
Scroll up to zoom out.
Scroll down to zoom in.
Hold the scroll to move the grids;

How to change difficulties:
1.Open the game, the default value of width is 10, the default value of height is 10 and the default ammounts of mines is 10;
2.Player can input the value they want in the input field and use the start level button to restart a level.

How to quit the game:
Alt+F4
